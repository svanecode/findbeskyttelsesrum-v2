# Product Brief

## Goal
Findbeskyttelsesrum v2 helps people in Denmark quickly find nearby shelters and understand how reliable the underlying information is.

## Core User Journey
1. Enter an address or use current location.
2. Review relevant shelter results.
3. Open a shelter detail page with source, status, and trust context.

## Product Principles
- Trustworthy: show source and freshness clearly.
- Calm: avoid noisy layouts and unnecessary UI.
- Simple: keep the main journey short and readable on mobile.
- Fast: prefer server-rendered content and small client islands.

## First Release Scope
- Homepage with address input, current-location action, and featured shelters.
- Placeholder pages for search, shelter details, municipality pages, data transparency, and admin.
- Supabase-backed shelter and municipality data with clear source tables and audit-ready overrides.

## Non-Goals
- Advanced ranking or geospatial search.
- Interactive maps.
- Full admin workflows.
- Generic backend abstraction layers.
